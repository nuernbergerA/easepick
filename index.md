# easepick
{: .no_toc .fs-9 }

Date picker - tiny size, no dependencies

## Features

- No dependencies
- Uses [Shadow DOM](https://developer.mozilla.org/en-US/docs/Web/Web_Components/Using_shadow_DOM)
- Supports plugins
